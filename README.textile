h1. GVIM 7.2 配置文件在Windows XP下的使用方法

  * 将本目录vimfiles整个拷入C:\Documents and Settings\magic\之类的目录中去，其中magic是用户名
  * 运行本目录中的setup.rb, 并根据提示安装MONACO.TTF字体

  =============================================

  * 如果有必要，去http://railsapi.com/下载帮助文档包解压到rdoc目录中，作为ruby的帮助
  * 如果有必要，修改上一级目录中的_vimrc中firefox的目录等
