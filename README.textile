h1. GVIM 7.2 配置文件在Windows XP下的使用方法

= 将ctags.exe拷到C:\Windows目录下
= 将findutils-4.2.20-2.exe与grep-2.5.4-setup.exe安装到C:\GnuWin32目录下
= 将firefox安装到C:/Program Files/Mozilla Firefox目录下
= 将本目录整个拷入C:\Documents and Settings\magic\之类的目录中去，
= 然后将本目录中的_vimrc与.irbrc文件单独拷贝到C:\Documents and Settings\magic\目录中即可
= .irbrc 要配合 gem install fastri才能在irb中快速查找
= 将Globals.rb拷到C:\ruby\lib\ruby\gems\1.8\gems\fxri-0.3.6\lib\复盖，改善fxri字体与大小

特别说明：

其中magic是用户名，请改为你自己的用户名。

如果你的firefox或findutils或grep安装的目录不一样，请自己修改_vimrc中的目录
