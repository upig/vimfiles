h1. GVIM 7.2 配置文件在Windows XP下的使用方法

  * 将本目录整个拷入C:\Documents and Settings\magic\之类的目录中去，
  * 然后将本目录中的_vimrc文件拷贝到C:\Documents and Settings\magic\目录中
  * 安装本目录中的MONACO.TTF字体（拷到控制面板的字体中去）
  * 去http://railsapi.com/下载帮助文档包到vimfiles目录内，解压到rdoc目录中
  * 在_vimrc中修改改$PATH, 确保其中包含本vimfiles目录, 以及firefox目录等

特别说明：
以上目录都是示例目录，比如其中magic是用户名，请改为你自己的用户名。
